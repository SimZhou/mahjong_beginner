## 工作方法

### 翻译流程
1. 使用 curl 下载原始 HTML（http://beginners.biz/）
2. 提取 main-contents 和 contents 区域
3. 保持图片路径为相对路径（../hai/, ../images/）
4. 使用 GLM-4 模型进行日文到中文翻译
5. 生成 Markdown 文件
6. 每 2-3 页推送一次到 GitHub

- 下载工具: curl
- 图片处理: 保持原始 HTML 的相对路径
- 翻译模型: GLM-4 (zai/glm-4.7)
- Git 推送: 自动提交并推送到 GitHub
