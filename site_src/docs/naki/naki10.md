# 混一色

混一色因为只要把一种花色和字牌集中起来就能做成，所以非常受初学者欢迎。

而且它不但容易做，打点也高，确实是最值得主动去狙击的役种之一。

但混一色真正拉开差距的地方，在于对鸣牌和牌形的理解。

实战里经常能见到：

1. 该鸣的牌没有鸣
2. 不该鸣的牌乱鸣
3. 结果把自己原本能和的手牌鸣坏了

## 不该鸣的手牌

混一色就算鸣牌也有 `2 番`，所以一般没必要强执着门清。

那么，什么情况下反而应该先不鸣？

像例 1 这种，门清本来就完全足以做到听牌。

如果一鸣就只剩下副露混一色，甚至还会顺手带上一气通贯这种额外价值，那就应该尽量拖到最后再鸣，先争取门前混一色。

顺便一提，这手只要万子进张，基本都会以两面以上的好形听牌收尾。

这当然是个偏极端的例子，但如果已经是门清混一色好形一向听，那就值得先靠门清努力看看。

很多人会不管这些，照样一路碰下去。

但只要出现死对子，这种时候更好的路线往往是门清混一色七对子。

这种时候，还是该先门清把形整理一段再说。

因为剩下来的牌形实在太苦了。

如果这时别人立直，你根本没有顶回去的力量，而且还额外失去了原本可以当安全牌的 ![西](../hai/sha.gif)。

这个代价很重。

## 该鸣的牌、该放过的牌

混一色里最重要的鸣牌原则，是：

**不要把手牌自己鸣断。**

一旦这样碰，![2m](../hai/man2.gif) 和 ![5m](../hai/man5.gif) 就会变得非常难用。

最后当然也可能碰巧做成混一色对对和，甚至跳满，但那只是“刚好成功”而已。

更多时候，这种鸣法只是在主动把自己鸣成愚形。

所以，这种会让自己制造愚形的鸣法，必须避免。

**不要什么都鸣。**
