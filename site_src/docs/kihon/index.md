# 麻将基础

这一栏处理的是整套教程里最底层的部分：先把麻将这门游戏在结构上看清楚，再去谈牌理、手役、防守和局况。

如果基础概念不稳，后面很多“为什么这么打”都会变成死记硬背。反过来，只要把这里的框架吃透，后面的章节会顺很多。

## 本栏内容

1. [麻将是什么](./kihon01.md)
: 先说明麻将到底是一种怎样的游戏，为什么它既受运气支配，又依然能拉开水平差距。

2. [麻将与概率](./kihon02.md)
: 讲清楚“牌效率”和“概率判断”为什么是做牌的核心，而不是先迷上某个漂亮役种。

3. [听牌与向听](./kihon03.md)
: 先把“距离和牌还有几步”这件事看清楚，理解什么叫听牌、什么叫一向听和两向听。

4. [面子与搭子](./kihon04.md)
: 理解顺子、刻子、边张、嵌张、两面的区别，建立最基础的牌形优先级。

5. [数牌与字牌](./kihon05.md)
: 说明不同牌种的功能差异，理解中张为什么强，客风为什么经常该早切。

6. [对子（1）](./kihon06.md)
: 讨论对子作为雀头时的作用，建立“雀头不必刻意先做、面子通常更优先”的基础观念。

7. [对子（2）](./kihon07.md)
: 讨论对子作为面子候补时的价值，理解为什么“两组对子最好，越多越臃肿”。

后续会继续补入：

1. 基本形与复合形
2. 麻将的听牌
