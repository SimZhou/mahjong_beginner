# 打牌选择思路

单纯只朝和牌前进、选择一手更有效率的打牌，本来不应该是特别困难的事。

上级者之所以能一眼判断出最有效率的一手，主要有两个原因。

1. 他们把大多数常见手牌都模式化了，知道哪一打能得到最宽的受入
2. 他们掌握了更好的比较方法

所谓“同样的配牌不会再来第二次”，这话当然没错。
但手牌一旦开始朝和牌方向整理，整体结构其实会逐渐落到一些高频模式里。
尤其是出现频率高的形，确实应该把正解记住。持续做出吃亏的选择，成绩当然会被拉低。

但我认为，比起单纯记住模式，更重要的是第二点，也就是**比较技术**。

现实对局里，不可能真的每一手都去逐项数：

1. 切 A 有多少种、多少枚
2. 切 B 又有多少种、多少枚
3. 切 C 还能变出多少好形

可即便不真的逐张数，仍然能靠更好的比较顺序，选出受入更宽的打牌。

我认为，这两点才是**牌理的真髓**。麻将战术书里，大家往往更爱谈“模式记忆”，但反而较少认真谈“如何比较”。

**防止失误，并且加快打牌选择的速度。**

单纯靠死记硬背，终究有上限。理想状态，是不管拿到什么手牌，都能自己推出正确答案。

我建议把牌理大致分成下面 4 类来思考：

1. 三向听及以下
2. 两向听
3. 一向听
4. 听牌

之所以把“三向听及以下”合在一起，是因为有两个原因：

1. 这类手牌光靠外观就判断向听数，本来就不太容易
2. 大多数情况都只是整理不要牌，没有太大必要按精确向听数再细分

## 1. 三向听及以下的基本牌理

手牌还在初期阶段时，最适合用的比较法是：

**切掉这张以后，会损失什么。**

这时没有必要过度执着于“下一手一定要降向听”。

当然也会有一些微妙选择，但对那些差距并不大的分歧，没必要花太多精力死抠。细效率固然重要，不过很多时候，保留手役可能、或者兼顾未来守备，反而更有价值。

## 2. 两向听的基本牌理

到两向听时，比起单纯比较“进一向听的受入枚数”，有时更应该关注：

**进入一向听以后，牌会不会更厚。**

麻将有一个非常重要的性质：

**越接近听牌，受入枚数通常越少。**

也就是说，真正最后的难关，其实往往是：

**一向听 -> 听牌**

如果在这一步受入太窄，就会直接让和牌变得很吃力。

因此，哪怕两向听阶段少掉一两枚受入，只要能换来一向听时更厚的牌形，通常也是值得的。

当然，也一直有一种意见认为“两向听时还是应优先进一向听受入”，这一点并不是完全没有争议。

不过总体来说，两向听常见的只是“还留着孤张”或者“搭子超载”这两类，处理难度还不算太大。

## 3. 一向听的基本牌理

到一向听时，最重要的评价项目就是：

**通往听牌的受入枚数。**

这里一两枚的差距不能小看。

当然，如果通往听牌的受入实在太窄，也会有一些情况宁可少拿一点当前受入，去保留好形变化。

而如果当前受入枚数完全一样，那么大多数时候，只要继续比较“好形变化有多少”，基本就足够应付了。

## 4. 听牌的基本牌理

听牌时，最重要的当然还是和牌牌数。

不过荣和并不是简单地“枚数越多就一定越容易和”。如果等待的牌本身就比较容易从场上打出来，那也属于好等待。

只是“牌容易不容易打出”很难精确量化，所以基本顺序仍然应该是：

**先看枚数。**

如果枚数没有差别，再去比手变枚数。尤其是单骑待，往往会带来很多手变。

## 这一页的结论

这一页讲的是总论。下一页开始，就会进入更具体的牌姿检证。

1. 上级者判断快，不只是因为记得住形，更因为会比较
2. 模式化的作用，是防失误和提速，不是把所有答案死背完
3. 三向听及以下，适合先看“切掉后损失什么”
4. 两向听时，要重视进一向听之后的厚度
5. 一向听时，听牌受入最重要
6. 听牌时，先看枚数，再看容易和到与手变
