# 按牌理打牌

所谓`牌理`，就是为了更高效地完成“四面子一雀头”而进行的结构判断。

它和“牌效率”几乎可以视为同一方向：不是先幻想役种，而是先判断哪张牌最不该继续留在手里。

它是整套做牌理论的根基。后面所有“该留什么、该拆什么、该不该贪手役”的问题，最终都会回到这里。

## 为什么牌理是做牌核心

把一局麻将先当成单局竞速来理解，本质上就是：

**四个人里，谁先更稳定地把手牌做成。**

而做成之前，手牌通常同时包含三种价值：

1. 速度
2. 打点
3. 守备余地

这三者都重要，但在多数普通局面里，`速度`优先级最高。因为：

1. 再高的手，和不到就没有收益
2. 你先和了，也等于把别人的潜在和牌一起打断
3. 对手还没听牌时，你也自然更不容易放铳

所以基础做牌最忌讳的一件事，就是为了手役的想象，把明显更快的路线主动放慢。

原页这里讲得很重：速度几乎可以同时覆盖打点和守备。你先和了，不只是拿到了自己的点数，也等于把别人本来可能和出的那一手拦掉了；你在别人听牌前结束牌局，也等于提前回避了将来放铳的风险。

所以“为了看起来华丽，故意把和牌巡目往后拖”这件事，通常是最不该做的。

原页还引用了一句很准的话：

**“表面上的华丽，只是在拖慢你真正和牌的巡目。”**

## 牌理的核心动作：列候选，再比较

这就是牌理最基本的做法：

1. 先把已经有明确用途的牌固定下来
2. 找出“还没有明确工作”的候选牌
3. 比较这些候选牌各自还能形成什么
4. 切掉其中功能最差的一张

所以，牌理并不神秘。它不是靠感觉拍脑袋，而是一个持续重复的动作：

**列出候选 -> 比较功能 -> 切掉最差的。**

原页把它说成两步：

1. 先把候选打牌列出来
2. 再逐张比较这些候选的工作能力

反复做这件事，切错牌的次数就会大幅下降。

## 为什么比较比直觉更重要

实战里很多错误，不是完全不会打，而是没有真正比较。

比如有的人会想：

1. `tyun` 是字牌，危险先留
2. `sou5` 靠近宝牌，感觉有戏
3. `man9` 是边张，也许能做幺九相关手役

这些理由单独听都像那么回事，但如果不把它们放到同一张桌子上比较，最后很容易留下了看似有故事、实则效率最低的牌。

真正有用的不是“我能给它编出什么理由”，而是：

**和其他候选牌相比，它到底是不是更有用。**

## 这一页的结论

1. 牌理是为了更快、更稳定完成手牌而进行的结构比较
2. 基础做牌里，速度通常是第一优先级
3. 不要先从役种幻想出发，而要先看当前牌形缺什么
4. 先固定已有用途的牌，再列出候选弃牌
5. 牌理的基本动作就是：`列候选 -> 比较 -> 切最差`
