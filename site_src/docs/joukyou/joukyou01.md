# 局势判断基础

## 什么是局势判断

初学者往往只会看自己的手牌。  
但随着巡目推进，牌桌上会不断给出各种信息。  
把这些信息一起纳入考虑，再去决定切牌、鸣牌、立直与否，这就叫做**局势判断**。

如果一直只盯着自己的手，很容易漏掉真正重要的信息。  
说得尖一点，那就只是“配对麻将”而已。

麻将当然需要很多技术，但其中尤其重要的一项，我认为就是局势判断。

连手作也会随着局势而改变。

例如，边张和坎张相比，通常麻将里都会嫌弃边张。  
但即便这一点，也会因为局势不同而反过来。

比如：

场上 ![3p](../hai/pin3.gif) **已经见了三张**

这种局面下，反而就会选择保留边张。

再进一步说，从这手里去切 ![3m](../hai/man3.gif) 或 ![4s](../hai/sou4.gif)，也都可能成立。

如果已经到了海底，那更不用说，当然是切最安全的牌。  
如果 ![3m](../hai/man3.gif) 是确定能过的牌，那就会直接切 ![3m](../hai/man3.gif)。

麻将正是因为有局势判断，才是真正的麻将。

局势判断里最重要的一块，其实是要不要对对手手牌进攻、要不要转守，  
也就是所谓押引。  
不过这件事放到最后一章再讲。  
这一章先集中讨论：**结合局势做手牌。**

## 两种局势

所谓局势判断，其实并不复杂。

只需要看两种东西：

**场况** 和 **点棒状况**

### 场况

所谓场况，就是牌桌本身的局面。  
例如：

- 宝牌是什么
- 四家的舍牌如何
- 谁已经副露
- 自己是亲家还是子家
- 现在是第几巡

这些全部都属于场况。

### 点棒状况

另一个同样重要的，就是点棒状况。  
因为麻将胜负最终是按整场结束时的分数来决定的，  
所以四家的点棒必须一直放在脑子里。

---

## 局势与战略

真正重要的，是根据上面这两种局势，先立出战略再去打牌。  
你平时打牌时，真的有先想过这一步吗？

例如，如果场上已经供托了 `4` 根立直棒，  
那就会自然形成一种战略：**尽快和牌，把这些立直棒拿到手。**

如果头名是亲家，那么另一种战略就会很有力：  
**去做更高打点的自摸手。**  
因为一旦自摸成功，还能靠亲被自摸进一步缩小分差。

也就是说，比起速度、效率这些局部问题，  
**首先应该先有战略。**

就算是这种起手，打法也完全可能不同：

- 如果是东场亲家，可以用九种九牌流掉
- 如果是断崖垫底，就能同时看国士和混一色，先从 ![4s](../hai/sou4.gif) 之类开始处理
- 如果是南四头名，就可以把国士骨架保留下来，做成随时可退的形

也就是说，同一副手牌，照不同战略去打，答案会完全不同。
