# 染手（1）

染手，尤其是混一色，是非常重要的手役。

门清有 3 番，副露后也还有 2 番，在打点面上属于明显被优待的手役。它当然有“容易被看穿”的缺点，但只要材料足够，就应该积极考虑。

它也是少数可以在配牌阶段就认真纳入判断的手役之一。

这一页讨论的核心，是只从手牌本身出发，判断要不要从配牌阶段去做染手。

本来染手还会受到三类条件影响：

1. 场况
2. 巡目
3. 点棒状况

但这一页先把条件固定为：东场平场序盘，不考虑额外局势压力，只看手牌本身。

索子足足有 `11` 张，而且形状也非常理想。

`1s` 是对子，`3s` 到 `8s` 之间又形成了连续连接，这几乎就是一手在说“请你去做清一色”的牌。只要不是终盘，一般都应直接朝清一色前进。

切牌上，这里应先切 `5m`。

按通常牌理，可能会考虑先切 `1m`，为未来摸 `6m` 留出万子两面变化；但像这种形，连万子的两面变化都已经没什么价值了，既然决定要染，外色里更危险、也更没用的那张就该先处理。

但并不是说，同色牌有 `10` 张以上就一定该染。

例 2 虽然筒子也有 `10` 张，但正解是老实切 `发`。

因为这手本来就适合走门清立直平和路线，后面根据摸牌，还可能自然带上断幺九、三色、一气通贯。也就是说，这是一手**不染也已经足够强**的牌。

既然如此，就没有必要为了“看起来快染上了”而硬把它拖去染手。

尤其是那种为了染手而主动拆掉已经做好的面子，基本都属于不该碰的打法。

### 理论小结

<span class="green">不必染的手，不要硬染。<br />主动拆一组现成面子去染，基本属于禁手。</span>

但麻将里总会有例外。

例 3 就是少数可以拆现成面子去染的情况。因为这手一旦染起来，打点会明显上升，而且形状也不算难和，此时可以果断拆掉筒子面子，转向染手。

不过这必须明确：这种情况只是少数例外。

在平场东场序盘，值得你为了染手而主动拆掉一组现成面子的手，实际上非常少。

## 2. 双役牌对子时，要尽量把染手抬到满贯线

### 例 4

<span class="b">例 4</span><img src="../hai/man2.gif" width="24" height="34"><img src="../hai/man4.gif" width="24" height="34"><img src="../hai/pin2.gif" width="24" height="34"><img src="../hai/sou2.gif" width="24" height="34"><img src="../hai/sou3.gif" width="24" height="34"><img src="../hai/sou4.gif" width="24" height="34"><img src="../hai/sou7.gif" width="24" height="34"><img src="../hai/sou7.gif" width="24" height="34"><img src="../hai/sou8.gif" width="24" height="34"><img src="../hai/ton.gif" width="24" height="34"><img src="../hai/ton.gif" width="24" height="34"><img src="../hai/haku.gif" width="24" height="34"><img src="../hai/haku.gif" width="24" height="34">　　摸到<img src="../hai/sou5.gif" width="24" height="34">　　宝牌<img src="../hai/man4.gif" width="24" height="34">

如果手里有两组役牌对子，就应尽量往染手方向拉，把整手往满贯线以上抬。

例 4 里虽然手中还有宝牌搭子，但当然还是应该瞄准索子的混一色。

不过这里还有一个很重要的实战细节。

很多初学者会直接从宝牌 `4m` 开始切，觉得“反正我要染手”。这种切法太过火了。

混一色最大的缺点，本来就是容易暴露。如果把坎张搭子直接从宝牌边开始拆，对手几乎立刻就会判断你在做索子混一色，警戒度会一下子抬高。

既然这手本来已经很好，就该尽量让自己的舍牌不那么显眼，因此这里应先普通地从 `2p` 切起，降低暴露度、提高和牌率。

而且就算染手是主线，后面也完全可能马上摸到 `3m`，路线并非从这一巡起就必须完全锁死。

### 理论小结

<span class="green">如果有两组役牌对子，就应尽量配合染手，把整手往满贯方向推进。</span>

## 3. 远距离染手时，守备优先

### 例 5

<span class="b">例 5</span><img src="../hai/man2.gif" width="24" height="34"><img src="../hai/man6.gif" width="24" height="34"><img src="../hai/man8.gif" width="24" height="34"><img src="../hai/pin5.gif" width="24" height="34"><img src="../hai/sou1.gif" width="24" height="34"><img src="../hai/sou2.gif" width="24" height="34"><img src="../hai/sou3.gif" width="24" height="34"><img src="../hai/sou5.gif" width="24" height="34"><img src="../hai/sou8.gif" width="24" height="34"><img src="../hai/haku.gif" width="24" height="34"><img src="../hai/nan.gif" width="24" height="34"><img src="../hai/nan.gif" width="24" height="34"><img src="../hai/pei.gif" width="24" height="34">　　摸到<img src="../hai/pin1.gif" width="24" height="34">　　宝牌<img src="../hai/pei.gif" width="24" height="34">

这里默认 `南` 是客风。

这手老实说是烂牌。如果是我，会先从 `5p` 这类中张开始切，保留索子混一色的远距离可能。因为就算按正常牌理推进，这手也很难真的和出来。

不过，这种远距离混一色和前面的“好材料染手”完全不是一回事。

如果你判断这手最后很可能需要弃和，那么甚至可以故意从中张开始切，让别人更容易感觉你在做染手，同时自己保留更扎实的守备余地。

从诈唬效果来说，副露当然会更显眼；但从守备角度看，这手并不该轻易鸣牌。

只要对手的手牌质量在平均以上，他们往往还是会正常推进。麻将里的诈唬效果没有想象中那么高。

所以这手比较合理的方针是：除非 `白` 或 `北` 再重，否则不必强求和牌。

很多初学者会在第一张 `南` 出现时就碰掉，但这正是要学会克制的地方。不是所有看上去“有役”的牌，都值得你一路硬冲。

有时更重要的心态反而是：

**这一手就算不和，也没有关系。**

### 理论小结

<span class="green">远距离混一色默认守备优先。<br />在真正看到和牌希望之前，不要勉强鸣牌。</span>

## 这一页的结论

1. 染手是少数可以在配牌阶段就认真考虑的高价值手役
2. 同色张数多且结构集中的牌，应积极往染手走
3. 不染也已经很强的手，不要为了染手强拆好面子
4. 只有在打点和和牌率都明显提升时，拆面子去染才成立
5. 两组役牌对子会显著抬高染手价值
6. 远距离染手默认守备优先，在真正看到和牌希望前不要乱鸣
