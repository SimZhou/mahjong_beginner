# 役牌

役牌最大的优点，就是**几乎没有额外限制**。

断幺九、平和这些常见役当然也很容易和牌，但它们会对整副十四张牌持续施加约束。相比之下，役牌只要把那组三张字牌凑出来，就能满足一番缚，而且副露后依然成立。

实际上，麻将里最常出现的手役之一就是役牌。谁更会处理字牌，谁更会把役牌和副露衔接起来，谁的和牌率通常就会更高。

## 1. 保留役牌

起手之后，很多人通常会按“客风 -> 役牌 -> 不要的数牌”的顺序切牌。

但如果想提高役牌手的利用率，更好的习惯往往是：在条件允许时，优先切真正没用的数牌，把役牌多留几巡。

当然，数牌的有效牌通常比字牌更多，所以前提是你得先认出哪些数牌确实工作能力最低。

## 2. 役牌还是断幺九

一般来说，断幺九整体上比役牌更有优势。

原因在于：断幺九在副露时更容易鸣，和其它手役也更容易复合，比如平和等路线都比役牌更自然。

### 理论小结

<span class="green">当最终打点相同时，确定的手役优于不确定的手役。</span>

## 3. 关于鸣牌

役牌基本上可以一鸣就碰。

“役牌要等二鸣”这类说法更像是旧时代的战术。按现在的环境，一鸣反而更符合主流速度观。

原因主要有三层：

1. 就算是便宜和牌，只要先和掉，也是在掐断别人的机会，这种价值往往不止牌面点数本身。
2. 在赤牌规则里，就算先碰成便宜手，后面摸进赤牌后依旧可能抬高整手价值。
3. 最关键的是，现在很多人都以速度优先，等到二鸣往往已经来不及了。

事实上，即使是上级者，也有很多一鸣役牌的场面。

不要被“这样不美”“应该再贪别的手役”这种旧价值观绑住。在现代麻将里，役牌一鸣就是基本思路。

不过，也不是所有役牌都该从第一张起立刻碰。

原页举了两个例外：

1. 手牌本身已经很完整，门清推进明显够强。
2. 手牌过于零散，就算先碰一组役牌，也未必来得及做成。

### 理论小结

<span class="green">役牌要等二鸣的情况，通常只出现在手已经非常好，或手已经非常差的时候。</span>

## 这一页的结论

1. 役牌最大的优点，是限制极少，而且副露后依然成立
2. 起手阶段若能先切真正无用的数牌，就应尽量给役牌更多重叠机会
3. 一般来说断幺九更有优势，但当断幺九尚未确定时，确定役牌往往更值得保留
4. 现代麻将里，役牌一鸣是默认思路
5. 只有在门清明显更强，或手牌过散导致先碰也没意义时，才更适合暂缓鸣牌
