比如这手牌，不能因为看得到 `234` 三色，就去打 `7s`。

因为只要摸进 `1m` 或 `5p`，三色就会散掉。

中途若去切这种不上不下的 `5m`，也一样谈不上好。这里更赚的做法，是先把索子的坎张拆掉，顺势往断幺九、平和这种更自然的路线走。

原页在这里强调得很明确：很多初学者总说自己不会做役，但实际更常见的问题反而是**太想做役**。尤其有人会硬往混一色、对对和去靠，结果副露了三次还没听牌，这种局面一点都不罕见。

麻将首先讲的是“**形**”。

先想清楚：

1. 四面子一雀头要怎么做出来
2. 最快的成型路线是什么
3. 在不明显牺牲速度的前提下，再让手役顺势附带进来

也就是说，顺序一定要是：

**先有牌形，再谈手役。**

## 手役的难度和点数，并不总是平衡

麻将里一共有将近 `40` 种役，但真正经常出现的，其实只占三分之一左右。剩下很多役，几乎可以当成“实战里用不上的役”。

最典型的就是三杠子。

这类役最根本的问题是：**难度和回报完全不平衡。**

不仅不值，甚至还可能因为乱开杠，多翻出几张宝牌，把别人本来只是立直一番的小手直接喂成倍满。

所以这种吃亏的役，完全没必要硬追。

## 常见手役才是真正该练的手役

原页这里引用了天凤站上的统计。我们直接看这张表：

| 役 | 出现率 | 等级 |
| --- | --- | --- |
| 立直<br>役牌<br>平和<br>断幺九 | `42.6%`<br>`38.7%`<br>`22.5%`<br>`21.2%` | `A` |
| 混一色<br>一杯口<br>三色<br>七对子<br>对对和 | `6.1%`<br>`5.0%`<br>`4.3%`<br>`3.0%`<br>`2.9%` | `B` |
| 一通<br>全带幺九<br>三暗刻<br>清一色 | `1.7%`<br>`1.2%`<br>`0.8%`<br>`0.8%` | `C` |
| 其他 | `0.5%` 以下 | `D` |

表的原始条件是：

**有食断的东南战，样本数在 `2` 万件以上。**

这个统计说明得很直接：

1. 麻将的手役虽然很多，但真正常做的就那么一批
2. 实战里最重要的，是把 `B` 级以上这些高频役练熟
3. 出现率不到 `0.5%` 的稀有役，基本可以先无视

### 理论小结

**手役的难度和点数，并不一定保持平衡。没有必要勉强去追那些难度高、却并不划算的手役。**

## 这一页的结论

1. 做役不能脱离牌形和速度单独讨论
2. 不要为了漂亮或稀有的役，主动放慢本来更自然的成型路线
3. 麻将里“难做”不等于“值得追”
4. 真正应该熟练掌握的是高频役，不是收藏型稀有役
5. 基本顺序始终是：先有牌形，再看哪些役可以顺势附带
