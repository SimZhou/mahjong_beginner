# 手役思路（2）

## 1. 手役是“形成出来的”

基本上，手役并不是“主动去指定”的，而是“做着做着自然形成”的。  
如果先在脑子里把最终形定死，然后把所有不符合那个终局的牌都切掉，这种打法就叫“定向打”。  
我认为，这种“定向打”最好绝对不要用。

麻将里，你无法自己决定接下来会摸到什么牌。  
摸牌和打牌人的意志没有关系。  
所以麻将的铁则，就是顺着摸牌来做手。  
定向打是大忌。

靠“定向打”真正占到便宜的场面，几乎没有。  
这种打法，更常见的只是把各种可能性主动掐死。  
手顺才是最好的打牌方式。

## 2. 手役的优点是什么

很多人会把手役的价值理解成“提高打点”，  
但在现在的麻将里，真正比收入的话，宝牌、赤牌、祝仪往往更强。

与其这么想，不如把手役看成：  
<strong>手役是为了让你能鸣而存在的。</strong>

---

**例 2**  
<img src="../images/01.jpg" width="370" height="127" alt="看似靠赤牌和出的例2牌形图" />

这手牌乍看像是“赤 3 幸运和出来了”的牌谱。  
但实际上，打牌的人是非常明确地在做手役。

看得出来，他连 <img src="../hai/ton.gif" width="24" height="34" /> 和 <img src="../hai/sou1.gif" width="24" height="34" /> 的对子都拆掉了，  
为的就是保留役牌。  
因为自己是北家，所以北是役牌；而东在南场只是客风。

在有筹码的赤牌规则里，这种手里有 2 张赤牌的牌，无论如何都很想把它和出来。  
但如果坚持门清去做，离立直又太远。

所以就给它补上断幺九或者役牌这个手役，  
把它做成可以鸣牌和牌的手。

“可以鸣”这件事，在赤牌麻将里非常重要。  
<span class="b">有手役 = 可以鸣 = 更快</span> 这个式子是成立的。

把手役也当成提速工具，这就是更现代的理解方式。  
赤牌麻将里断幺九之所以重要，也正是因为它让你能自由地鸣。

## 3. 手役的缺点

不同手役的差别当然存在，  
但最大的问题，还是：  
它会减少受入枚数。

虽然追三色会让和牌打点提高，  
但这手牌里，显然还是听牌机会最多的切 <img src="../hai/man4.gif" width="24" height="34" /> 更好。

当手役（打点）和速度必须二选一的时候，  
<span class="b">多数情况下，优先速度会得到更好的结果。</span>

手役还有另一个缺点，就是它会限制你能使用的牌种。  
比如宝牌是筒子时，你去做索子混一色，那宝牌就用不上。  
做全带幺九时，摸到赤牌反而会很难受。  
又比如七对子听牌时被立直压制，自己再摸到那张对子牌，  
很多人都经历过那种左右为难的感觉。

要记住：  
手役有时也会在速度和守备上拖后腿。
