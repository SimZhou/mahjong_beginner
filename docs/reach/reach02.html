    立直还是默听（1）

    上一页讲过，立直的优点远大于缺点，所以基本可以先记住一句话：

    听牌以后，默认先立直。

    真正应该选择默听的情况，反而比较少。

    因此更实用的学习方式，是先把“哪些情况明显更适合默听”记住。只要不符合这些例外，通常就直接立直。

    判断立直还是默听，主要看两件事：

    1. 手牌本身：听牌形状、打点、改良空间
    2. 局面状况：点差、巡目、他家动向

    这一页先只讲“平场、手牌本身”的判断。也就是默认东场、点差平稳、巡目不深、其他人也没有明显动作的情况。

    1. 最容易判断的默听场景

    例1

    像例 1 这种役满听牌，就是最容易判断的默听场景。 因为立直不会再带来更高打点，反而白白把自己已经听牌的信息送出去，所以没有必要立直。

    例外基本只有“自摸四暗刻”这种情况。那种时候立直的意义，不是为了加番，而是为了逼退对手、拉长巡目、提高自摸率。

    2. 默听已经是跳满、倍满以上时

    例2

    如果默听本身就已经是倍满这种大牌，通常也不应该立直。 原因很直接：

    1. 立直再加 1 番，未必真的还能跨到更高一档
    2. 但一旦立直，荣和率通常会下降
    3. 这么高的打点，本来就已经足够作为决定手

    所以，默听已经是跳满、倍满以上时，基本优先默听。

    等待差、染手容易被看出来的场合，这个结论会更明显。

    3. 满贯附近是分水岭

    如果一手牌默听已经有满贯附近的打点，立直和默听之间就会出现灰区。

    通常可以先这么记：

    1. 默听已经是满贯确定：更偏向默听
    2. 还没到满贯，只是立直后有望涨到满贯：更偏向立直
    3. 好形、巡目早：立直倾向更强
    4. 愚形、改良多：默听倾向更强

    也就是说，满贯大致是一个分界点。 低于满贯时，立直带来的收益提升通常更明显；高于满贯时，立直的边际收益就开始下降了。

    例3 宝牌

    像例 3 这种默听 7700 的手，即使已经不小了，但如果等待很好，仍然完全可以积极立直。

    4. “高目高、低目低”的手，往往反而更该立直

    例4 宝牌

    像例 4 这种“高目 7700、低目 2000”之类的听牌，通常更应该立直。 因为你既可以争取高目，又不会因为低目和牌就完全没有收益。

    无论高目是外筋还是内筋，都不是核心问题。核心是：低目和牌也还能接受，而立直后整体收益会上升。

    5. 愚形的大物手，多半优先默听

    例5 宝牌

    像例 5 这样，已经有役、又是 8000 点、而且还是愚形，通常就是默听更有利。

    因为这种牌一旦立直，荣和率往往会下降；而默听本身已经足够大，没有必要为了不确定的一发、里宝牌，再去牺牲稳定性。

    理论 / 总结

    1. 役满确定：无条件默听
    2. 默听跳满、倍满以上：优先默听
    3. 满贯附近：看等待质量与巡目
    4. 默听未满贯：通常立直收益更高

    你可以先把这个判断顺序记住。后面几页再继续细化“好形、愚形、改良、局况”这些更具体的分支。
