手役的思考方法(1)

麻将有魅力之一的是役作り。
役满开始难度高的役，容易役的话无论谁都想。

[一二万 一二三四五六七八饼 一七饼 一七九索] 摸[八饼]

但是，瞄准过度也是思考。

[二三四五万 三三三四五饼 一一三七饼 一二三六七饼] 摸[六饼] 宝牌[三饼]

例如这个手，二三四的三色能瞄准，所以[七饼]切是不存在的。
如果引[一饼]或[五饼]的话三色崩坏。

中途半端的切[五饼]也不怎么好。

索子的嵌张丢弃，比瞄准听牌得点。

初学者说手役不能经常，但瞄准手役过度失败的人多。
特别是牌理效率本位和向听的人多。
三副露还没有听牌也不少。

麻雀首先是"形优先"。
首先考虑"四面子一雀头怎么做"，
不牺牲速度，范围内瞄准手役的东西。

麻雀有40种类以上的役，其中经常出来的役是3分之1左右。
剩下的使不用的役认为就好。那个最多的是三暗刻。
难易度和报酬完全不成比例。
不如那样，鸣增加宝牌，让只有立直的手变成满贯也有可能。那样的损失役无理瞄准没有的必要。

麻雀的役虽然很多，但能经常作的役是限定的。
在这个表，出现率 B rank 以上的手役的技术重要，
0.5%未满的 C rank 以下的役几乎无视就好。

理论·总结

手役的难易度和点数的关系，绝不是平衡的东西。
无理瞄准难度高的手役没有的必要。
