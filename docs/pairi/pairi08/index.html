初期牌理(1)

三向听以下的手牌，牌理的难点是手役的价值应该考虑什么程度。
这是因为如果考虑鸣的话，进张变少，认为到听牌的速度可能逆转。

这里为了结论变得明确，在纯粹牌效率上议论。

作为初期牌理，首先举出浮牌有几枚阶段的手牌。

例1 [二二三万 二六七饼 一五五六七八索] 摸[八饼] 切[南南]

如果有孤立牌的话，比较残留的孤立牌的功能比较好。

在这里 [七万][二饼][一索][南南] 是都不对面子·搭子·对子当的孤立牌。
功能最低的牌是什么的话，是[南南]。

[南南]只有引[南南]做成对子·刻子的力。

能做成顺子的数牌的价值高是明白的。

但是在这里切的牌只有[南南]。

另外切[一索]也没有那么大的损失。

"[二三索] 引，而且 [四索] 不引的情况"是失去1个面子，所以切[一索]的理由哪里也没有。

例2 [二三四六八万 一二三四五饼 一四五六七饼] 摸[四万] 切[一一]切

切[北]也没有完全问题。

但是这个手是 [二索] 或 [三索] 无论引哪个都是切[一一]的手。

也就是和字牌一样，除此之外没有使用的方法。

如果是那样的话，如果下巡早立直的话，可以丢弃[北]，重叠的话之后可以作为安全牌丢弃2张，若干虽然有的但[北]作为有价值的牌吗。

理论·总结

孤立字牌是最难成为面子的部分，
如果有能成立役的风牌，切它是绝对正确的。
但是，如果有相当不要的数牌，
也有考虑将来的守备，从数牌切的手。

例5 [一一三五八万 五六七七八索] 摸[三索] 切 [一索]切

相同数牌的比较的话，应该残留连续形。

牌理来看，孤立老头牌比对子·搭子价值低。
