一向听的牌理(3)

比较"哪个选择更接近和牌"，培养比较的意识。

验证残留的2个搭子的完成待的一向听。

例1 [一二三四五六七饼 一一三三四七饼] 摸[二饼] 切[一索]

两面听残留2个的最多见的一向听形。
像这样的手牌，通过残留双碰进，可以扩大听牌机会。

听牌机会对比

  听牌机会 进张枚数
切[一索] [一饼][二饼][四饼][九饼][六索][九索] 20枚
切[二饼] [一饼][四饼][六索][九索] 16枚

通过残留索子的好形形，有一气通变的可能，
但即使考虑打点，认为不是有利的打法。

如果双碰进的牌变薄的话，
也和安全牌交换好。

例2 [一二三四五六饼 一一二三四五六饼] 摸[二饼]

切[二索]

切[二饼]和切[二索]，都是一向听23枚相同。

所以比较听牌时的待。

变化A

[一二三四五六饼 一一一二三四56饼] 摸[九饼]

变化B

[一二三四56饼 一一一34567饼] 摸[九饼]

如果保险的双碰进听牌的话，
切[二索]的话变成三面听。

虽然小差，是实战中频繁的形，所以应该机械地打。

例3 [一一二三四567饼 一一一234567饼] 摸[九饼]

切[五索]和切[五索]都是15枚相同，

所以比较听牌时的待和进张的差。

变化A有利是明白的。

变化B的话，听牌时的[一饼][三饼]双碰，
切[五索]的话，[一饼][五饼][三饼]双碰。哪个好不知道，
但进张的差，变化A的切牌是全部有效的，所以变化A有利。

变化C

[一一2饼34567饼 一一一2饼345678饼] 摸[三饼]

所以切[二饼]正解。

但感觉上 [五索] 是差的判断，
实际上切[四饼]的人多。

这也作为理论记住。

例4 [一二饼 一一一二三四饼] 摸[六饼]

切[四饼]

变则三面听含的手牌。
嵌[五饼]进有，但[四饼]切的话，听牌时的待
[五饼][二饼][五饼] 双碰（待牌差没有）
[五饼][五饼][二饼] 变成三面听（待牌差有）
所以切[五饼]。

例5 [六七饼 一一一23456饼] 摸[八饼]

手牌拍子切[八饼]的话不只是。
看一通是恶手。

有嵌[七饼]进，所以切[四饼]。

理论·总结

双碰进是弱搭子，残留的话手宽。
变则三面听含的手牌，考虑嵌进的待比三面听好的情况。
