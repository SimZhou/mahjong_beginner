打牌选择的思考法

单纯只考虑朝和牌的打牌选择，
那么不是那么困难。
上级者，是在第一手就能判断效率最高的打牌的人。

为什么能瞬间判断呢。
可以举出下一个2个理由吧。

（1）几乎手牌模式化，把握最广进张的打牌

"相同的东西不会来2次"的配牌，
如果朝向和牌整理手牌的话，手牌在一定程度上模式化了。
应该记住出现率高的形的正解。
如果继续错误的选择，会影响成绩。

（2）掌握优秀的比较方法

"切A的话8种8枚，切B的话…"
实际上数枚数是不现实的。
即使不那样，选择进张广的牌。

（1）（2）是"牌理的真髄"我认为。
在麻将战术中，（1）被注目，
为什么（2）几乎不被说。

我认为是这样。
从组合的数来说，到暗记有极限。
无论什么样的手牌，理想是能导出正解。

作为牌理的方法，我是

（1）三向听以下
（2）两向听
（3）一向听
（4）听牌

分为这4模式对应比较好。
为什么三向听以下要作为1个总结。那有2个理由。
三向听以下的手牌，第一眼判断几向听是困难的，
而且几乎是不必要的整理，所以按向听数分类的必要不太有

1. 三向听以下的基本牌理

初期阶段的手牌，"切这个牌的话会变成几向听"比较比较好。

不需要专心于降低向听数。

微妙的选项也有，
但没有必须深想那么没有大差的选择。

细小的效率差也重要，但考虑保留手役的可能性
考虑将来的守备，我认为重要的多。

2. 两向听的基本牌理


比较一向听的进张枚数也很重要，但如果有1～2枚的差的话，取一向听时的进张多的情况有利。

作为麻将的重要性质有"越接近听牌，进张枚数变少"

这样的东西。

最后的难关是"一向听　→　听牌"

这里进张枚数少的话，和牌变得困难。

因此即使稍微减少两向听的进张，重视一向听时的厚度也是好的。
（不过"重视一向听的进张枚数"的见解也很强。）

两向听是孤立牌残留的模式，搭子超过的模式多切牌的话没有那么困难。

3. 一向听的基本牌理

听牌的进张枚数成为最重要的评价项目。

这里的1～2枚的差无法挽回。

不过，如果听牌的进张怎么也狭窄的话，有减少向听的进张保留好形变化的情况。

如果进张枚数没有差，
比较好形变化的枚数几乎可以对应。

4. 听牌的基本牌理

当然和牌牌的枚数最重要，但荣和牌单纯枚数多并不容易和牌。场里容易出的牌如果是待牌的话，可以说是优秀的待。
不过"待的容易度"相当难数值化，所以第一考虑枚数是基本的。

如果枚数没有差，比较手变化的枚数也很重要。
特别是单骑待手变化多。

以上是概论。
