两向听的牌理(2)

下次，一面子和雀头完成，四搭子块的模式。
没有浮牌的情况，选择有2个。
（1）丢弃功能最低的搭子。
（2）把搭子块变成单独搭子，维持四搭子。

最近的研究，（1）的打法被认为是强力的，
本网站也把（1）作为基本的打法。

那么用实际的手牌看吧。

例1 [五六七七九饼 一二三四五六七八饼] 摸[八万]

切[二索]

索子的搭子是[三索]二度进，所以毫无疑问应该丢弃这里。

切[三三]是一向听的进张减少的恶手。

例2 [三四五六七饼 一一一三四五六七饼 一二三四五索] 摸[八万]

切[五索]

比较不构成搭子块的[三万]和[五索]的价值。

只是变成连续形万子优秀，但索子
有[二索][三索]的两面听，引[四索]的话能变成一向听，所以没有残留浮牌的意思。345的一通稍微残留。

引[六索]的话，
[一索][四索][七索]
的二度进，万子是劣等的。

例3 [四七八饼 三三八八饼 四五七八八九索] 摸[五饼]

切[九索]

比较不构成搭子块的[四万]和[九索]的价值。

[七八索][八索][九索]
构成连续形，
但是做面子的力远不及内测的中张牌。

例4 [二二三六八饼 一一一三四五饼] 摸[八万]

切[一一]

相当微妙的选择。

对子多，[一一][四]的价值没有。
牌理来说[一一][二二][三三][四]和[三万]的差几乎没有。

所以这里取手役的差。

万子即使做两面听也不能变成一通，
索子的话引[二万]～[五万]能做一通。

这里对子多，所以保留[二万]的手是充分有的。

例5 [一二三四饼 二二三四五六789饼] 摸[七万]

切[三万]

像例5那样的三对子形，不应该残留双碰进。

浮牌是
[七饼][六饼]

但两边残留切[三万]。

不吝惜1张损失的话，我认为保留好形变化好。

还有，故意残留功能低的牌的手也有。

例6 [三三四饼 一一二三四五饼] 摸[北]

切[七饼]

像例6那样，两面听齐全的情况会相当。

即使残留[七饼]，现在也没有比[七饼]更强的搭子。
如果切了可能变成和牌的牌，应该先切。

理论·总结

"一面子和雀头完成＋三搭子"的两向听是，
　从残留的浮牌中，选择最不要的东西是基本。
　手牌根据不同，把3组搭子块变成单独搭子，
　全部残留浮牌好。
　这个模式的几乎全部，是三对子形中的两面听决定。
