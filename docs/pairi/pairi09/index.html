初期牌理(2)

上次是浮牌比较能决定的手牌。
这次是搭子比较，或者是孤立牌和搭子比较的模式。

1. 六搭子一雀头形

这个模式，单纯做搭子比较。

例1 [一三六七饼 一一二五五万 二二三四五六七索] 摸[五万] ⇒ [一一]切

搭子比较的定石，丢弃边张。

摸[三万]的话，即使立直，
丢弃[一一][三三]更好，
所以从外丢弃有利。

[一一三六七万 二五六七八索] 摸[三万]

例2 [二三六七八饼 一一二三四五饼 二三四五六七索] 摸[五万] 切[四索]或者切[五万]

全部是两面搭子，所以丢弃二度进的索子搭子。

丢弃[七八饼]不只是三色，

作为最终的待 [三万]￣[六万] 比 [六万]￣[九万] 优秀。

这里需要注意的是
切[一一]的话变成四向听。
如果丢弃唯一的雀头，毫无疑问会提高向听数。

即使看混一色，从例2的形
切[一一]不能说是恶手。

例3 [二三四五饼 二二二三三五五七饼 一二三四五六七索] 摸[三饼] 切[二万]

例3是全部的颜色都有二度进搭子。

"切这个牌的话，损失的牌是什么"
如果考虑的话，自然而然就明白了。

万子是两面搭子所以当然残留。

饼子和索子比较的话，残留[六饼]摸切正解。

如果[六万][九万]的话，三色目（混一色一通）
所以切[九万]是正确的。

理论·总结

六搭子形是丢弃损失最少的搭子。
丢弃唯一的雀头是法则。

2. 五搭子　＋　浮牌

基本上比较残留的孤立牌就好。

例4 [二三五七八饼 三三五六九饼 二三四六七八索] 摸[四万] 切[五万]

万子做二面子的话，[五万]是几乎没有持的意思的牌。

[六索]有残留[二索][四索]的价值。

切[九饼]作为选择的话，被称为"混一色病"。

即使混一色消失2种安目进张，在1个面子也没有完成的状态下一口气朝向立直是相当危险。

例5 [一三五七万 三七八八九索 一二二三四五七七索] 摸[三索] 切[九万]

搭子的数来说有5个，[七饼]的嵌边张二度进是太苦了。
像这样有恶形搭子的情况下，残留孤立牌的选择是充分有的。

例6 [四七七饼 一一三四五六七饼] 摸[八万] 切[一一]或者切[二万]

例6也是，饼子是[三饼][六饼][九饼]
变成三度进嵌张的价值相当低。残留浮牌的话，那之后的手牌推进似乎会顺畅。饼子的一通瞄准进张被限定过于现实。

理论·总结

五搭子形基本上切价值最低的浮牌就好，
如果有二度进嵌张等困难搭子的话，
丢弃搭子残留浮牌有利的情况多。

3. 搭子超过形

例7 [四七七饼 一二三四五六七饼] 摸[四万] 切[七万]

关于搭子超过的初期牌理，重点是牌效率以外的东西。

例7有[九万]和[七万]的搭子，[九万]的进张广。

但是从[九万]的进张中，[七万][四万]是好形变化。

从[七万]的进张中，[四万]是坏形变化。

所以切[七万]是正解。
切[九万]是恶手。

理论·总结

关于有多个好形变化的搭子的处理，
好形变化的枚数多的搭子应该残留。
进张少，好形变化多的搭子应该丢弃。

4. 搭子超过　＋　浮牌

例8也是二度进塔子的情况，但和例7不同，浮牌的价值相当不同。

浮牌[二万]作为孤立牌的价值低。

即使保留[二万]，[二万]的进张宽，好形变化也丰富。

丢弃[二万]的话，[九万]的嵌张进变窄，进张减少2种。

所以例8残留[二万]是正解。

理论·总结

浮牌的进张宽，好形变化丰富，比有二度进的搭子价值高的情况下，
应该优先保留浮牌。
