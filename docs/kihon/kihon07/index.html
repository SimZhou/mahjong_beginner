对子（2）(対子(トイツ) ２)

考虑对子作为面子候补。

对子的性质

对子有2组时作为面子候补的价值最高这样的性质。

举个具体例子吧。

【例1】

例如像例1这样只有1组对子时，

只有2张无法期待，而且即使引来了雀头也会消失。

（ 虽然是有效牌，但向听数不变）

【例2】

但是像例2这样变成2组对子的话，
对子就开始作为面子候补发挥作用了。

与例1相比，
向一向听的进张增加了 的4张。

有2组以上对子时，
每组对子各有2张进到刻子的进张。
这些进张被称为双碰进张，或者双碰。

【例3】　　摸

那么像例3这样变成3组对子会怎么样呢。

即使再增加一组对子，也只是增加双碰进张的2张。
从例3打破嵌搭的话会损失4张，
所以再增加对子是不利的。

从例3来看，从牌理考虑答案是切 。
切掉哪组对子最好，将在第2章考虑。

（这里知道3组对子不利就可以）

双碰因为每组只有2张进张，
如果有4组以上对子，手牌的进张就会变窄。
因此，会更多地使用"碰"来推进手牌或者瞄准七对子。

理论·总结

对子有2组时作为面子的作用最强。
（双碰的效果很大）
随着对子增加，作用变弱，面子手变得难做。
