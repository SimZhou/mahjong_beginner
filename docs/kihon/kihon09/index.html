3枚复合形

3枚复合形只有
"对子＋搭子的复合形"、"嵌张2个的复合形"
这2个而已。

对子与搭子的复合形

这是在对子的项也稍微提到过的。
是搭子的1张重叠的形，与只有搭子的进张相比
增加进到刻子的进张2张。

复合的种类 具体例 完成面子的牌 枚数
边张 +　对子 6枚
嵌张　+　对子 6枚
两面听　+　对子 10枚

即使有这样的形，轻易固定为对子或搭子的打牌人很显眼。
对子固定是4张或8张（两面听的情况）的损失，
搭子固定是2张或4张的损失。

【例1】　　摸

初学者不知为何倾向于喜欢对子。

从这样的形也有人会切 ，但这是消除了 的4张进张，否定饼子变差的不好打法。

2张或4张左右，可能感觉没什么大不了。
但是听牌速度胜负要素很大的麻将，
如果只是单纯减少进张的话明显是损打。

切 是标准的打法。

【例2】　　摸

从例2这样的形先打 这样的打法，在大多数情况下也是恶手。将现状只有16张的进张减少到12张是很大的损失。

而且这手牌是断幺九，碰听（碰之后的听牌）也有效，

只要不是 很难切的场况，就是切 的一手。

理论·总结

对子与搭子的复合形不要太早固定为对子或搭子，
扩大进张是基本的打法。

一向听中留下不要牌的搭子固定也会损失4张，所以要避免。

两嵌

嵌张2个连接的形被称为两嵌。

　　　　　　　　

两嵌有以上5种形式，
全部是嵌张的2倍，有8张进张。

两嵌如果只是进张枚数的话和两面听一样是8张，可以说是好形。但是与其他牌组的配合，两嵌会变得不自由。

【例3】　　摸

例3是留下两嵌切 最接近听牌，但会放弃一通这一手役。如果从索子的两面听填补的话，听牌也变成了普通的嵌张。

【例4】　　摸

例4是无论切什么都会产生损失的困难选择。

哪种形在打牌上都会迷惑吧。
两嵌因为使用3张牌，随着接近听牌手牌会变得拘束。
当然这也是无法避免的部分……

理论·总结

嵌张的二连形被称为两嵌，被认为是仅次于两面听的好形。特别是在序盘发挥威力，但接近听牌时比两面听相当劣等的形。
