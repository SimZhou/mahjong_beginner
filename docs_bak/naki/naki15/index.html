    瓷砖铺贴技巧(3)

    方牌的最后一招是方牌，不是为了攻击自己的手牌，而是为了阻止对手的手牌攻击。

    有些人在使用这些技巧时表示不舒服，所以

    实战中使用时请务必小心。

    一枪被抹掉

    如果方形牌在站立后立即放置，则一击的权利消失。

    一枪命中的可能性不是很高，所以

    没必要冒险出手。

    反之，如果完全没有风险，最好一次性消除。

    在上图所示的情况下，不可能去拿起笔。

    并擦除一次。记住它是个好主意。

    梗阻

    如果歌和一辉重叠，一辉基本上优先。

    （如果口吃者明显说得更快，则有安排优先考虑口吃者。）

    因此，你也可以用牌来挡住对手想要口吃的牌。

    移动海底，消除海底

    海底（ハイテイ：その局最後の牌）でツモられた場合は１ファンアップです。

    終盤立直が入った場合、海底を消せるのであれば消しておくのが無難です。

    尤其是阿加里连索的情况下，只要将父母的结印抹去一次，就有一定的效果。

    另一方面，即使很难进入最后阶段十，

    增加口吃并试图恢复的人的 Tsumo 数量是个好主意。

    不会去阿加里的剑

    不小心就会被当成初学者，但是

    了解曲的隐藏好处有很多好处。

    例如，假设下议院正在上升，而最后的攻击选择是水下瓷砖。

    如果你有木头要制作，请保留它并在最后一轮时制作。

    通过这样做，海底瓷砖将移动一位，让您可以擦除直立人的 Tsumo 编号和海底。

    杠すると相手に宝牌をのせてしまうので

    自分が鳴いている場合極力杠を避けたほうが良いとされます。

    しかし、現状ラス目であれば大明杠でもしたほうが良い状況があるのです。

    通过玩杠，你不仅可以期待杠宝牌，还可以

    随着情况变得通货膨胀，其他房屋成为大玩家，

    也有可能另一个人会击败那个大玩家并提升排名。

    即使你排名第二，如果你的对手以微弱优势排名第一并且是你的父母。

    当最后一投有站立时，击中的策略是有效的。

    目的是在家长的帮助下提高排名。

    不过我手里有足够的安全牌，

    这也是你不必担心最后一个回合被逆转的时候。

    当条件满足时，这样的迫击炮也是一个考虑因素。
