    3片式复合型

    三片复合形式为 “对子 + 杠子的复合形式” “两个杠子的复合形式”

    只有这两个。

    对子と搭子の複合形

    これは对子の項目でも少しふれておきました。

    即使是这种形状，也有很多情况可以很容易地将其固定在贝子或通子上。 修复时损失4~8件（以梁门为例） 固定底座需要损失2或4块。

    [示例 1] Tsumo

    出于某种原因，初学者往往更喜欢刺子。

    从这样的形状 我看到有些人剪切 这是一种糟糕的玩法，因为它消除了四张牌的接受度并否定了引脚发生不良变化的可能性。

    2 或 4 张卡似乎没什么大不了的。

    不过，作为一个速度比拼成分较大的武将，

    如果单纯降低接受度，显然是一种失败的策略。

    标准的打字方式是剪切。

    在许多情况下，使用示例 2 中所示的形式不是一个好主意，其中 。如果将目前仅接受的16张卡减少到12张，将是一个巨大的损失。

    koji 和 koji 的复合形式并没有太早固定为 koko 和 koko， 基本策略是扩大接受度。
