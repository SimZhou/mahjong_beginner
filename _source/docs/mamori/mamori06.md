# 吸入步骤(1)

涉水有明确的程序，所以一定要仔细遵循说明。

应大幅减少转移。

## Yanwa的基本思想

安全度の高い牌から切っていく

这是原则。
 
我不在乎我的手是否会碎掉。

因此，“实物”、“完全安全的瓷砖”等3块瓷砖在现场

只要有这种事，我就砍掉。

从肯定通过的瓷砖上进行切割非常重要。

第一轮实际金额有可能增加，

有时其他家庭会动摇，而一个人会重新站起来。

即使有安全瓷砖，为什么要先把“看起来安全的瓷砖”切掉呢？

从防守的角度来看，这是一场“失败”。
