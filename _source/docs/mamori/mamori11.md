# 扣牌（1）

上一节讲了对副露手的基本防守。
本节讲更主动的防守手段：**扣牌**。

扣牌的意思是：
有意识地把对手想要的牌“扣在手里不打出”，
从而阻断其副露推进。

## 1. 扣牌的意义

扣牌的核心作用是限制对手进攻速度。

当你判断某家是当前主要竞争对手时，
不让其轻易副露，往往比盲目自顾自推进更有价值。

它常用于：
- 防止对手快速听牌
- 防止对手通过副露把小手变成可和手
- 控制关键局面的点差变化

## 2. 扣牌的代价

扣牌不是“免费动作”，会有明显副作用：

1. 你自己的手牌效率下降
2. 你可能因此失去先制机会
3. 你只压制了一家，另外两家反而更轻松

所以扣牌不能滥用。
如果只是“想压人”而没有明确收益，
往往会变成亏损操作。

## 3. 什么时候值得扣牌

### （1）目标已经明确在副露推进

没人副露时，不要凭空“防幽灵手”。
扣牌应建立在看得见的威胁上。

### （2）目标是你的直接竞争对手

例如南场中后段，你与某家点差接近且名次关键。
这时压制该家速度，收益会明显高于平时。

### （3）可控风险范围内可执行

如果为了扣牌要长期抱大量危险牌，
那本质上接近“自我僵死”，得不偿失。

扣牌应在“还能维持基本安全与牌效率”的前提下进行。

### （4）优先针对上家（吃的方向性）

对“吃”而言，最能直接控制的是上家。
对非上家，只能部分影响“碰”，控制力明显更弱。

因此扣牌优先级一般是：
- 上家明确染手/副露路线时优先考虑
- 非上家只在关键牌（如役牌、宝牌周边）上选择性扣牌

## 4. 实战原则

至少满足下列条件中的两项，再考虑扣牌：
- 目标已副露推进
- 目标是名次竞争对手
- 你能在不崩手牌的前提下执行
- 扣牌方向上有实际控制力（通常是上家）

否则，不如按正常效率与防守原则处理。
