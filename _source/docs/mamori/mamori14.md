# 读立直（1）

“读立直”并不是要精确猜中对方唯一听牌，
而是要把牌分成两类：
- 明显更危险
- 相对更安全

这才是实战可执行的读牌方式。

## 1. 不追求“唯一答案”

同一副舍牌，可能对应多种听牌形。
因此不要执着“对手一定在等哪一张”。

真正有用的是：
- 哪些牌在这些可能性里反复出现（高风险）
- 哪些牌被多种可能共同排除（相对安全）

## 2. 五的早切（高风险信号）

当某花色的 `5` 很早切出时，
往往意味着该花色两侧延伸结构被提前放弃，
由此可推导出若干高风险筋。

经验上，这类“5 的早切 + 立直前牌河结构”
常对应特定筋风险上升。

注意：
- 这不是绝对命中
- 只是提高危险度权重

## 3. 四间筋要看“成立条件”

四间筋本身是危险提示，但必须结合形成过程。
如果舍牌顺序不支持该结构，就不能硬套。

换句话说：
- 不是“看见四间筋就必危”
- 而是“先判断是否像四间筋手顺，再提升警戒”

## 4. 两面搭子落的筋

有些立直前手顺表现为“先舍两面搭子的边牌”，
这会让对应筋变得更危险。

其逻辑是：
- 打手不愿保留效率差的受入
- 在效率取舍后，留下的等待区间会向特定筋集中

这类读牌适合在中后盘、信息较多时使用。

## 5. 远位横向切牌的跨筋

若某色牌从外侧向内侧连续处理，且中间有跳步，
有时会形成“横切跨筋高风险”结构。

该模式条件复杂，不建议生搬硬套。
实践中只需记住：
- 当手顺异常顺滑且目标色连续推进时
- 跨筋危险度应上调

## 小结

读立直的核心不是“猜中全部”，而是“排序危险度”。

建议实战流程：
1. 先现物与字牌兜底
2. 再用筋/壁做第一层筛选
3. 最后用手顺信息提升或降低个别牌风险

这套方法稳定性远高于“拍脑袋猜听牌”。
