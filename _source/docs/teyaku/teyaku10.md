# 对子手（2）

本节讲七对子与面子手的分歧判断，
以及七对子听牌时的等待选择。

## 1. 七对子 vs 面子手：怎么选

基本原则：
- 若两条路线向听差不多，通常面子手更稳
- 若七对子明显更快（例如已接近听牌），可转七对子

不要“先入为主只走一条路”，
应根据进张动态调整。

## 2. 转向七对子的常见信号

以下信号出现时，可提高七对子优先级：
1. 对子数量已足够且结构均匀
2. 面子手受入狭窄且改良慢
3. 局况需要快速成听而非高上限

反过来，若面子手仍有宽受入两面主线，
通常不必强转七对子。

## 3. 七对子听牌的等牌选择

七对子常见误区是“只看剩余枚数”。
更实战的做法是同时看：
- 枚数
- 对手是否容易打出
- 该牌在当前河里是否显眼

一般来说：
- 场上已见、但仍可能被打出的字牌/老头牌，有时更实用
- 过于显眼、明显被人回避的中张，实战荣和率可能偏低

## 4. 刻意选择“可荣和”等待

七对子并不总是追最大理论受入。
在关键局面里，
主动选择“更容易被打出”的等待，
往往比死守理想枚数更有效。

## 小结

对子手章节的核心是：
- 动态比较七对子与面子手
- 听牌后按“枚数 + 出和率”选等待

把这两点结合，七对子就不再是“碰运气役”，
而是可控的实战路线。
