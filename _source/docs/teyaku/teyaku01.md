半吊子地切 <img src="../hai/man5m.gif" style="display:inline;vertical-align:middle;margin:0 1px;" width="19" height="26"/> 也不算好。

这里更合理的思路，是把索子的坎张拆掉，老老实实往断幺平和方向走。

很多初学者会说“我总是做不出手役”，但实际更常见的问题不是“不会做”，而是**想做的太多，结果把速度和形状都牺牲掉了**。  
尤其是硬做混一色、对对和的人特别多，常常三副露了还没听牌。

麻将首先是“形先行”。

要先想的是：

1. 怎么做出四面子一雀头
2. 在不明显牺牲速度的前提下，再顺手争取手役

而不是反过来，先决定要做什么役，再勉强把牌往那个方向扭。

## 常用手役其实并不多

麻将里接近有 40 种手役，但真正经常出现、值得重点掌握的，只占其中大约三分之一。  
剩下很多役，实战里几乎可以视为“不常用役”。

最典型的就是三杠子。  
难度和报酬根本不成比例。甚至还可能因为开杠送出新宝牌，反而把别人的立直小手抬成大牌。

这种明显不划算的役，完全没必要硬去追。

## 看看实际出现率

原站引用了天凤站上的统计，给出了带食断的东南战样本结果（两万局以上）：

|---|---:|---|

| 役牌 | 38.7% | A |
| 平和 | 22.5% | A |
| 断幺九 | 21.2% | A |
| 混一色 | 6.1% | B |
| 一杯口 | 5.0% | B |
| 三色同顺 | 4.3% | B |
| 七对子 | 3.0% | B |
| 对对和 | 2.9% | B |
| 一气通贯 | 1.7% | C |
| 混全带幺九 | 1.2% | C |
| 三暗刻 | 0.8% | C |

麻将里的手役很多，但真正应该反复练、重点掌握的，其实是表里 B 级以上这些高频手役。

那些出现率连 0.5% 都不到的稀有役，基本可以忽略。

### 定式总结

手役的难度和打点回报，并不总是成正比。  
没有必要勉强去追高难度手役。
