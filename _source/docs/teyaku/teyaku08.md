# 三色同顺

三色同顺在现代实战里不是“每手都硬做”的役，
但它仍是非常重要的 2 番路线。

关键不在于执着做三色，
而在于：
- 该做时能看见
- 不该做时能及时放弃

## 1. 滑动（Slide）

三色最基础手筋是“同序数在三门里滑动替换”。

例如原本做的是某门普通顺子，
摸进相关牌后可替换成三门同序，
把普通效率手转成三色路线。

这是最常见、最实用的三色起点。

## 2. 双天秤

所谓“双天秤”，
是同时保留两种三色可能（如 678 与 789）。

优点：
- 可根据后续进张灵活选路线

缺点：
- 常会牺牲当前受入或速度

因此只在“局况允许、改良价值高”时使用，
不要当成固定公式。

## 3. 利用余牌（边缘牌）

很多看似多余的牌，
其实是三色的“种子牌”。

判断标准：
- 这张牌是否能连接成三门同序
- 保留它会不会显著伤害当前效率

若代价不高，可以短暂保留观察一巡。

## 实战原则

做三色时按这条顺序判断：
1. 先看受入是否明显受损
2. 再看三色完成率
3. 最后看局况是否需要提打点

当“速度损失小、打点收益大”时，三色值得追。
反之就应及时回归高效率主线。
