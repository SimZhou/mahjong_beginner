# 平和

平和是最常见、最实用的门清役之一。
它的本质是“效率手”：
- 全顺子结构
- 非役牌雀头
- 两面听和牌

## 1. 核心思路：尽量做两面

平和手的关键不在“凑役名”，而在“持续保留两面”。

常见判断：
- 同等条件下，优先能转两面的结构
- 边张/嵌张若可替换，应尽早替换

## 2. 两嵌变两面（两嵌形利用）

某些两嵌相连的形状，
一旦进张可迅速转成高质量两面。

这类形往往比“看起来顺眼但僵硬”的牌型更有价值，
因为它兼顾了速度与最终听口质量。

## 3. 对子落与平和路线

实战里经常出现：
- 保留对子可稳一点
- 但落对子后可进入平和高效率路线

何时该落对子：
- 对子是役牌/价值低且阻碍两面时
- 落对子后受入明显扩大时

## 4. 不要只看当下一巡

平和手是“连贯效率手”，
应至少看两步后：
- 这巡切什么
- 下巡摸进后是否仍能维持两面扩张

## 小结

平和打法可以记成一句话：
**优先两面，减少低效率搭子，保持门清连续进张。**

做到这点，平和不仅是一个役，
更是整套牌理效率的基本功。
