# 和平（和平（和平））

和平是最容易创造的角色之一。

如果你能有效地合拢双手，你的双手自然会变得平静。

## １．両面への変化


平和は全てシュンツで構成され、かつ最終的な听が両面でなくてはいけません。

両面をつくっていくことが、平和手作りのポイントです。
