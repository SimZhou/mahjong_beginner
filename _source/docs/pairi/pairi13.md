# 一向听牌理（1）

一向听阶段的牌理，是手作中最关键的技术之一。
因为它会直接影响和牌率。

远离和牌的手牌，效率差一点问题不大；
但临门一脚时的失误，往往是致命的。

## 1. 基本思路

一向听时可先按下面的流程判断：

最难的是“特殊模式”：
即使是受入最多的打牌，也仍不能算“足够好”的情况。

## 2. 基本模式

“是否足够好”没有绝对标准。
我个人把“受入16枚以上”作为及格线。

这个门槛看起来高，但即使有16枚受入，平均也要7-8巡才听牌。
一局只有18巡，实际上已经很紧。

### 本节总结

1. 先比较受入枚数。  
2. 受入接近时比较听牌质量。  
3. 形不足时优先考虑好形变化能力。  
4. 必要时可接受“退向听”来换取整体 EV。
