# 一向听牌理（4）

本节讨论“无雀头一向听”的典型判断。

结论先行：
- 多数情况下，优先做成“雀头未固定 + 两搭子保留”
- 不要过早把某个对子固定为雀头

## 1. 无雀头形的核心思路

无雀头一向听的优势在于弹性。
一旦过早固定雀头，常会损失大量受入。

因此常见优先顺序是：
- 保留高效率两搭子
- 避免把强搭子硬拆成低效率结构

## 2. 暗刻在场时更应保持弹性

当手里有暗刻或高价值块时，
固定雀头的机会成本更高。

因为暗刻本身已经提供结构稳定性，
其余部分更该追求“受入最大化”。

## 3. 什么时候可以固定雀头

以下场景可考虑提前固定：
1. 你需要立刻成听（后盘/抢速度）
2. 固定后几乎不损失受入
3. 固定后能显著提升听口质量或打点确定性

如果不满足这些条件，
一般仍以“雀头不确定形”更优。

## 4. 愚形两搭子并存时

若最后两搭子都偏愚形，
应优先选择“听牌机会更多”的那条线，
并在后续摸牌中继续争取改良。

不要为了想象中的理想最终形，
在一向听阶段过早牺牲受入。

## 小结

无雀头一向听的实战关键是：
- 不急于定雀头
- 先吃受入，再谈完美形

这会显著提高你从一向听到听牌的稳定率。
