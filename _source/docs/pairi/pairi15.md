# 一向听牌理（3）

本节继续讨论一向听阶段的牌理取舍。
核心不是“看起来哪种牌型漂亮”，而是：

1. 听牌机会总量
2. 听牌后的听口质量
3. 打点与速度的平衡

## 1. 先比较听牌机会总量

当两种打法都能维持一向听时，
优先选择能让下一步听牌机会更多的一方。

尤其在中盘以前，
“多几张受入”通常比“勉强保留某个高打点梦想”更有价值。

## 2. 愚形改良与好形优先

若当前一向听包含明显愚形（边张/嵌张），
而另一种打法能提高改良概率，
通常应优先改良路线。

原因是：
- 听牌速度更稳定
- 听牌后和牌率更高
- 被先制后仍有应对空间

## 3. 暗刻与对子结构的处理

当手里有暗刻或对子堆积时，
不要过早“固定雀头”。

很多情况下，保留雀头不确定形，
反而能扩大听牌机会。

但若局面已进入后盘，
且你更需要“尽快成听”，
则可接受较保守的固定打法。

## 4. 打点路线何时让位给速度

如果高打点路线需要明显牺牲听牌机会，
且当前局况并非必须逆转，
应让位给速度路线。

只有在以下情况才应明显偏打点：
- 点差要求高打点逆转
- 你已明显后手，普通和牌价值不足
- 高打点路线并未大幅降低听牌率

## 小结

一向听阶段可用这条顺序：
1. 先看受入总量
2. 再看听口质量
3. 最后看打点需求

按照这条顺序，能避免多数“牌型好看但实战亏率高”的误判。
