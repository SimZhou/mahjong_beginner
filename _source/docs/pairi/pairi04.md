# 搭子理論(1)

## １．基本牌理

Toko的评价项目如下5分。

**(1) 接受的牌数（可以完成面的牌数）**

**（2）更改为更好的形状（三面、双面、两面等）**

（3）打点（涉及角色类型和宝藏板块）
 
（4）当你处于昏迷状态时，是多么容易醒过来。

这次，我们将评估最基本的（1）和（2）。
