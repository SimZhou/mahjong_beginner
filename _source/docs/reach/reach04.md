# 立直还是默听（3）

上一节讲了基本判断，这一节补充“边张/嵌张等愚形听牌”的处理。

## 无役听牌时

先看“是否无役且必须靠立直才能和牌”。
这种情况下，立直通常是默认选项。

但也有例外：
- 手牌改良空间很大
- 巡目还早
- 立直后打点提升有限

这时可以先默听观察一巡，等待改良。

## 愚形听牌的基本立场

即便是愚形，只要有宝牌或红宝牌支持，
立直收益往往仍然可观。

换言之：
- 无役愚形 + 有宝牌：倾向立直
- 无役愚形 + 无宝牌：更偏向默听

## 什么时候默听更好

以下情况可优先考虑默听：

1. 手牌可改良种类多（例如3种以上有效改良）
2. 改良后可变成两面或明显提高打点
3. 局况允许等待（巡目早、对手攻势不强）

这种“先默听后改良再立直”的路线，
常比立刻立直一手低打点更优。

## “拆听再做”也是选项

有些听牌看似已成型，但质量太差：
- 打点低
- 听口差
- 后续发展被锁死

这时不一定要在“立直/默听”二选一里硬选，
也可以选择“先拆听”，把手牌做回更高质量的一向听。

这是很重要的高级技能，尤其在平场局面。

## 有役听牌时

如果已经有役，默听可以直接荣和，
就不必把立直当唯一出路。

大体可按以下经验处理：
- 默听已是 5200 或更高：优先默听
- 默听只有 2600 左右：多半立直

当然最终仍要结合：
- 巡目
- 点数状况
- 对手速度与副露信息

## 小结

“立直还是默听”不是固定答案。

建议按这条顺序判断：
1. 是否无役、是否必须立直
2. 愚形质量与改良空间
3. 立直带来的增益是否足够
4. 当前局况是否值得冒险

当你能稳定做出这套判断，
立直章节的实战胜率会明显提升。
