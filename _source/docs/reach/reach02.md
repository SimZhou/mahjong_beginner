# 站立还是库罗？ (1)

前回立直の利点と欠点を挙げましたが、圧倒的にメリットの方が大きく、**「听牌したら立直するのが基本」**と思って差し支えありません。

ヤミテンを選択するケースの方がレアです。

したがって、ヤミテン有利のケースを覚えておいて

該当しない場面では、迷わず立直を打つようにするのが良いでしょう。

判断这是真话还是谎言有两点。

手牌和情况。 （麻将中的所有决定都是基于这两个因素做出的。）

**手板**：根据手板材质判断，如手板的扣板、圆点等。

**情况**：根据当前积分、其他玩家动向、轮次等判断。

首先，我们根据手中的牌来进行判断。

鸟羽比分形势平平，

轮班一结束，其他家族就不会再有任何动静。
