# 立直还是默听（2）

这一页继续讨论“立直还是默听”，重点放在**好形听牌**上。

所谓“好形”，就是更容易和牌的听口。最典型的是两面、多面张；另外像字牌双碰、字牌单骑这种，在实战里也常常比纸面枚数看起来更容易和出。

以前常有人说“平和 nomi 就默听”。理由是：1000 点的牌，还要先掏 1000 点立直棒，不划算。

但现代牌理已经基本否定了这种看法。因为**立直平和的平均和牌打点，明显不止 2000 点**，加上一发、里宝牌、门清自摸和压制对手的效果，整体收益是正的。

所以像例 2 这种情况，切 <img src="../hai/sou1.gif" style="display:inline;vertical-align:middle;margin:0 1px;" width="24" height="34"/> 或 <img src="../hai/sou4.gif" style="display:inline;vertical-align:middle;margin:0 1px;" width="24" height="34"/> 之后，都应该立刻立直。

## 3. 不要为了单一改良，放弃当下的好形听牌

**例3**

这类牌看起来会让人心痒：只要摸进某一张特定牌，就能变成三色、一气通贯之类的大改良。

但问题在于，这种“刚好只差 1 张”的改良，命中概率本来就不高。而你在等那张改良牌的时候，更常发生的事，是原本的和牌张先出来了。

如果为了等一个很窄的改良而选择默听，通常是得不偿失的。因为立直本身带来的 1 番、一发、里宝牌和对局压制，往往比你空等那张改良牌更实际。

例 4 比例 3 稍微复杂，改良种类更多，看起来更有“等等看”的诱惑。

但即便如此，还是不建议因为这些改良而放弃立直。原因在于：

1. 你不一定真的能先摸到改良牌
2. 就算摸到了，最终也不一定刚好和到高目
3. 立直本身已经提供稳定的增益

尤其像“平和 + 宝牌 1”或“断幺九 + 平和”这种 2 番手，本来就是立直后收益提升最明显的一类。

例 5 这种牌，改良确实很多，所以如果你想默听，也不是完全不能理解。
但它之所以会让人纠结，往往是因为一种常见误区：执着于所谓“最终形”或“最高形”。

问题是，实战里大多数时候你根本走不到那种最漂亮的完成形。更常见的情况，是你在此之前就已经和牌了，或者根本没等到改良。

所以，和其空想一个理想终点，不如先抓住眼前这副已经非常不错的好形听牌。

### 理论 / 总结

1. **好形低打点**：平场基本一律立直
2. **平和 nomi 也要即立直**
3. **不要为了很窄的改良去放弃当前好形听牌**
4. **“最终形”执念通常弊大于利**

一句话总结就是：

**好形听牌，默认立即立直。除非你有非常明确、而且很强的默听优势。**
