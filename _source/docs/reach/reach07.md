# 立直取听（1）

从这一页开始，进入“已经听牌后，应该取哪种听口”的问题。

原站把取听判断拆成 3 个顺序，这个顺序非常实用：

1. **枚数**
2. **打点**
3. **出和难易度**

## 判断 1：先看枚数

如果两种听口的和牌牌数有明显差距，优先选枚数更多的一边。

### 理论 / 总结

听牌后的取听顺序，建议固定成这样：

1. **先看枚数**
2. **再看打点**
3. **最后看出和难易度**

另外，要特别记住：

**当牌理差不多时，双碰听往往比嵌张、边张更实战。**
