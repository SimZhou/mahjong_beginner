# 立直理论（1）

这一页不讲具体局况，先整理几个最实用、最容易直接执行的立直基础原则。

## 1. 基本原则：该立直时就立刻立直

最常见的初学者错误之一，是：

1. 听牌后先默听转几巡
2. 看看有没有好改良
3. 然后突然摸切立直

这种打法表面上像是在“灵活观察”，实际上大多数时候都亏。

像例 1 这种听牌，看上去似乎还有手变到三色的可能，于是很多人就想“先转两巡看看”。

但这种等待通常不划算。虽然它有 <img src="../hai/pin4.gif" style="display:inline;vertical-align:middle;margin:0 1px;" width="24" height="34"/>、<img src="../hai/man1.gif" style="display:inline;vertical-align:middle;margin:0 1px;" width="24" height="34"/> 这类改良，但你在等改良的过程里，也完全可能先被别人打出你当前不能和的牌。

一旦如此，你不仅错过荣和，还白白损失了一发和压制效果。

所以，只要你的判断是“这手总体该立直”，那就不要再拖。

## 2. 立直 / 默听要在听牌当下判断清楚

像例 2 这种牌，也应该在听牌瞬间就把判断做好。

如果你决定立直，就当场立直；如果你决定默听，就继续默听，除非之后局况真的发生了明显变化。

最亏的，就是那种：

1. 先默听
2. 安目见逃
3. 又转几巡
4. 然后再突然立直

这种操作通常没有任何稳定的期望值优势。

更别提有些人还会把这种行为美化成“等一发时机”。
其实立直越晚，你的一发机会只会越小，这根本不成立。

### 理论 / 总结

**听牌时就把立直 / 默听判断做完。**

1. 判断结果是立直：马上立直
2. 判断结果是默听：只要局况没变，就继续默听

## 3. 判断立直时，要以“自摸和牌”作为基准

很多人一看到不好出的牌，就会先想：

**“这张不容易出吧？”**

这种思路太消极，而且经常会错。

即便是跨宝牌、或者看起来不太容易出的两面听，只要整体听口还不错，就照样可以大方立直。
因为你不是只靠别人打出来；**你还可以自摸。**

只要是两面听，自摸本来就有充分可能。判断时，如果只盯着“荣和难不难”，就会系统性低估立直的价值。

像这种更不好出的边张、而且还是宝牌周边的等待，也不能因为“看起来不容易出”就默认默听。

如果你立直后自摸，依然能拿到不错的打点。只要收益整体是正的，就该立直。

单骑听牌以外，等待牌“看起来容不容易出”，通常都不是判断立直与否的第一要素。

真正更稳定的判断方式，是：

1. 先看等待牌枚数
2. 再看立直后的综合打点提升
3. 最后才考虑“可能比较难出”这种因素

### 理论 / 总结

1. **不要拖巡再立直**
2. **立直 / 默听要在听牌当下决定**
3. **判断时以自摸和牌为基准，而不是只盯荣和**

一句话说完就是：

**该立直，就马上立直；判断立直时，要把自摸收益一并算进去。**
