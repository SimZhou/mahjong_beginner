# 立直理论（2）

这节讲几个容易在实战里直接用的细节。

## 1. 立直判断与“做牌判断”不完全一样

平时做牌时，常会优先“内侧效率、外侧延伸”。
但一旦已经进入立直判断阶段，
需要把“和牌速度与被读牌风险”一起考虑。

也就是说：
- 最优牌理切法，不一定等于最优立直切法。

## 2. 优先保留能兼顾打点与和率的结构

如果某种取舍能同时带来：
- 基本和率不下降
- 还有里宝牌/暗刻等增益可能

那通常优于“只追单一改良”的打法。

## 3. 少给对手额外信息

立直后河牌信息会被对手重点利用。
因此在可选范围内：
- 尽量避免把等待形暴露得太明显
- 让对手更难精准读到你的听口

“信息少”本身就是一种进攻价值。

## 小结

立直阶段要同时看三件事：
1. 和率
2. 打点上升空间
3. 信息暴露成本

当这三者冲突时，优先选综合收益最高的方案。
