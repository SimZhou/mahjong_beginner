# 局况判断的基础

## 什么是局况判断

新手往往只盯着自己手牌，但巡目推进后，场上信息会越来越多。
把这些信息纳入决策，去决定打牌、鸣牌、立直与否，这就是局况判断。

只看自己手牌，很容易漏掉关键情报。
说得直接一点，就会变成“只会拼图”的麻将。

麻将技术很多，但局况判断是最关键的技术之一。
手作方向会因局况显著变化。

## 两类核心局况

其实不复杂，重点就两类：

1. 场况。
2. 点棒状况。

场况包括：

1. 宝牌是什么。
2. 四家舍牌结构。
3. 谁在鸣牌。
4. 亲家/子家关系。
5. 当前巡目等。

点棒状况同样关键。
麻将以终局点数定胜负，所以四家点棒必须持续关注。

## 局况与战略

真正重要的是：根据这两类局况先立战略，再做打牌细节。

例如场上已有4根供托立直棒，
那就应该主动制定“尽快和牌把供托吃掉”的战略。

又比如当前第一名是亲家，
此时做高打点自摸的战略通常更有效，
因为成功后能利用“亲被自摸多付”更快缩小分差。

所以不是先谈速度或效率，而是先有战略。

哪怕是这种配牌，不同局况也会导向完全不同打法。比如：

1. 东场亲家可选择九种九牌流局。
2. 末位时可兼看国士与混一色，先从 <img src="../hai/sou4.gif" style="display:inline;vertical-align:middle;margin:0 1px;" width="24" height="34"/> 一带切起。
3. 若是南4局第一名，可保留国士方向作为随时退守手段。

这就是“同一手牌，不同战略”的核心。
