# 麻将入门教程 (Beginners Luck 中文版)

欢迎来到“Beginners Luck”日麻入门教程中文版！本网站致力于提供全面、易懂的日本麻将基础及进阶战术指导，内容完全翻译自 [http://beginners.biz/](http://beginners.biz/)。

不论你是刚刚接触麻将的新手，还是希望提升水平的初级玩家，都能在这里找到适合你的战术和牌牌知识。我们保留了原网站的说明图片，以确保您可以直观地学习所有的麻将牌理和牌牌攻防战术。

## 章节概览

1. **[麻将的进阶基础](kihon/kihon01.md)** - 了解麻将本质、概率、面子搭子、复合型及听牌牌牌的理论。
2. **[牌理与牌效](pairi/pairi01.md)** - 麻将中最核心的进张、浮牌、一向听牌牌和牌牌两向听牌牌的打法选择。
3. **[麻将的役种](teyaku/teyaku01.md)** - 常见的平和牌牌、断幺九、染手、三色及对子手的做牌思路。
4. **[宝牌与红宝牌](dora/dora01.md)** - 如何利用宝牌，以及含有红宝牌的现代麻将打法。
5. **[鸣牌 (副露)](naki/naki01.md)** - 碰、吃、杠的判断标准与食断、后付等技巧。
6. **[立直判断](reach/reach01.md)** - 为什么立直是现代麻将最强武器，如何选择立直或默听牌牌。
7. **[防守与弃和牌牌](mamori/mamori01.md)** - 安全牌、筋牌、壁牌的判断，以及完全弃和牌牌与兜牌技巧。
8. **[局势判断](joukyou/joukyou01.md)** - 根据点数和牌牌巡目调整战术，尤其是南场和牌牌最终局的打法。
9. **[攻防判断 (推与退)](osihiki/osihiki01.md)** - 何时该全攻，何时该撤退，衡量得失的铁则。

> 祝您在这个网站能快速提升麻将实力！
