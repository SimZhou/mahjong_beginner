# 3片式复合型

三片复合形式为
“对子 + 杠子的复合形式” “两个杠子的复合形式”

只有这两个。

## 对子と搭子の複合形

これは对子の項目でも少しふれておきました。


即使是这种形状，也有很多情况可以很容易地将其固定在贝子或通子上。
修复时损失4~8件（以梁门为例）
固定底座需要损失2或4块。

### セオリー・まとめ

杠チャンの二連形をリャン杠といい、リャンメンに次ぐ好形とされる。
特に序盤では威力を発揮するが、听牌が近づくにつれリャンメンよりかなり劣る形となる。
