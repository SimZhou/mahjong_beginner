# 3片式复合型

三片复合形式为
“对子 + 杠子的复合形式” “两个杠子的复合形式”
只有这两个。

## 对子和汼子的复合形式

我在关于 Toko 的部分谈到了这一点。


即使是这种形状，也有很多情况可以很容易地将其固定在贝子或通子上。
修复时损失4~8件（以梁门为例）
固定底座需要损失2或4块。

### 理论/总结

koji 和 koji 的复合形式并没有太早固定为 koko 和 koko，
基本策略是扩大接受度。

避免在象棋中用不必要的牌固定棋盘，因为这也会导致损失 4 张牌。

### 理论/总结

杠长的双式称为两杠，据说是仅次于两门的第二好式。
前期的威力尤其强大，但随着墙扑的临近，它的威力就明显逊色于梁门了。
