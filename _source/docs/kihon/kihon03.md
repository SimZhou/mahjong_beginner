# 听牌与向听

学习麻将手牌制作中的重要概念。
与其先考虑役种，不如先以牌型来理解手牌，这是核心。

## 麻将的和牌形

麻将无疑是完成和牌形为目标的游戏。
这个和牌形大致可以分为3种模式。

（1）完成四面子和一对子
（2）完成七对子
（3）完成国士无双

（2）和（3）是特殊的和牌形，基本是（1）的四面子形。
面子有刻子（コーツ）和顺子（シュンツ）的2种方式。

## 听牌（テンパイ）とは

再有一张就能完成和牌形的形状叫做听牌。
如果有役种就可以和牌。

【示例1】<img src="../hai/man2.gif" /><img src="../hai/man3.gif" /><img src="../hai/man9.gif" /><img src="../hai/man9.gif" /><img src="../hai/man9.gif" /><img src="../hai/pin5.gif" /><img src="../hai/pin6.gif" /><img src="../hai/pin7.gif" /><img src="../hai/sou2.gif" /><img src="../hai/sou2.gif" /><img src="../hai/haku.gif" /><img src="../hai/haku.gif" /><img src="../hai/haku.gif" />

<img src="../hai/man1.gif" />或<img src="../hai/man4.gif" />可以和牌的形状。

麻将不听牌就无法获得得分。（例外是只有别人的荣和）
另外，如果是门清（没有鸣牌的状态）就可以立直。
因此，尽可能快地到达听牌状态很重要。

## 向听数（シャンテン数）

【示例2】<img src="../hai/man2.gif" /><img src="../hai/man3.gif" /><img src="../hai/man9.gif" /><img src="../hai/man9.gif" /><img src="../hai/man9.gif" /><img src="../hai/pin5.gif" /><img src="../hai/pin7.gif" /><img src="../hai/sou2.gif" /><img src="../hai/sou2.gif" /><img src="../hai/sou3.gif" /><img src="../hai/haku.gif" /><img src="../hai/haku.gif" /><img src="../hai/haku.gif" />

距离听牌还差一手的状态。
<img src="../hai/man1.gif" /><img src="../hai/man4.gif" /><img src="../hai/pin6.gif" /> 摸到了就会听牌。

像这样再有一张有效牌就能到达听牌的状态
叫做**「一向听（イーシャンテン）」**

可以到达听牌的接收牌（在这个情况下是<img src="../hai/man1.gif" /><img src="../hai/man4.gif" /><img src="../hai/pin6.gif" />）
叫做**「听牌机会（テンパイチャンス）」**

听牌机会（牌的枚数）越多就越容易听牌，
可以认为是优秀的一向听。
