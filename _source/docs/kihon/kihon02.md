# 马一般与概率

说到Asashi的手工制品，
有些人可能认为这是塑造什么样的角色的问题，但是
我认为这与阿修罗的本质脱节。

手工工艺品的精髓在于创造“四面童子、麻雀头”的造型。

如果你在一场比赛中看阿修，

这是一款比赛，看四名玩家中谁能最快完成四面子一帕图。

もちろん最終的には持ち点で評価されますが、
宝牌（红宝牌を含む）が多く、役种がなくとも高得点が出る現在の麻将のゲーム性を考えれば
やはり本質は「役」ではなく「形」です。

由于只有先完成表格的人才能获得积分，
速度是不可避免的要求。

那么怎样才能快速适应呢？
这背后的想法是平铺效应和概率。
