# 马一般与概率

说到Asashi的手工制品，
有些人可能认为这是塑造什么样的角色的问题，但是
我认为这与阿修罗的本质脱节。

手作りの本質は「四面子・一雀頭」という形をつくることです。
如果你在一场比赛中看阿修，
四人中谁会最先完成“四面子一帕图”？这是一场你竞争的游戏。

当然，最后会根据你的分数来评价你，但是
考虑到目前《朝日》游戏的性质，宝藏图块较多（包括红色宝藏图块），即使没有角色类型也能获得高分。
毕竟，本质不是角色而是形式。

由于只有先完成表格的人才能获得积分，
速度是不可避免的要求。

那么怎样才能快速适应呢？
这背后的想法是平铺效应和概率。
