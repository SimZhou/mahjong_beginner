# 鸣牌技巧（1）

本节讲“鸣牌后的手牌管理”。
很多人会鸣得很快，但鸣后手牌很快变僵，
真正差距就在这里。

## 1. 鸣牌不是目的，和牌才是目的

鸣牌后要立刻问自己三件事：
1. 现在最快听牌路径是什么
2. 这条路径的听口质量如何
3. 有没有更稳的替代路线

如果鸣完后这三点都不清楚，
通常说明这次鸣牌本身就值得反思。

## 2. 先处理“无效拥堵块”

鸣牌后常见错误是：
- 继续保留看似有价值、但实际上互相冲突的牌块

正确做法是尽快清理拥堵块，
把手牌收束成“可直达听牌”的结构。

## 3. 注意读牌可见性

副露会暴露你的路线。
如果你的弃牌又进一步暴露等待，
对手就能更容易回避你的和牌牌。

因此在可选范围内：
- 尽量减少“把等待写在河里”的弃牌顺序
- 保持等待方向的隐蔽性

## 4. 不要只盯眼前一巡

鸣牌后如果只看当前一巡最优，
容易陷入“短快但死听口”的局面。

应至少看两步：
- 这巡切什么
- 下一巡常见进张后怎么处理

能提前两步规划，副露手质量会明显提高。

## 小结

鸣牌技巧的本质是“鸣后管理”：
- 快速收束结构
- 控制信息暴露
- 保持后续两步可操作性

做到这三点，鸣牌才真正从“加速”变成“增胜”。
