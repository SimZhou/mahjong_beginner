# 混一色

混一色因为“只收一种花色加字牌”这一点很直观，所以很受初学者欢迎。
而且它不难做，打点也高，确实是非常值得瞄准的手役。

但混一色真正拉开差距的地方，在于：

1. 什么时候该鸣。
2. 什么牌该鸣，什么牌不能乱鸣。
3. 是否理解鸣牌后剩余形状的质量。

## 不该鸣的手牌

混一色就算副露也有2番，因此没必要死守门前。
但这不代表任何时候都该鸣。

虽然例子稍极端，但如果是“门混一色的好形一向听”，完全值得继续门前推进。

### 本节结论

别什么都鸣。
混一色副露的核心不是“听得快”，而是“鸣完以后，形还要活着”。
