# 杠（2）

本节讲“有杠材也不该杠”的典型场景。

## 1. 手牌明显落后时，不要盲目杠

当手牌整体速度慢、结构差时，
贸然开杠常会把局面进一步推向失控：
- 给全场增加宝牌收益
- 自己未必吃到杠收益
- 反而提高被先制压制概率

这类局面应先修手效率，而不是追杠。

## 2. 局势不利时，杠可能是负收益

若你当前点数领先或需要控局，
开杠等于主动增加波动。

在“稳局优先”的场景，
除非杠后收益非常明确，否则应克制。

## 3. 杠后受入下降时应慎重

有些暗杠会让手牌受入明显减少，
即便账面打点上涨，也可能让和牌率下滑。

判断要点：
- 杠后是否削弱关键搭子
- 是否失去好形听牌机会
- 是否把可改良手变成僵硬手

## 4. 场上已有立直时更要保守

通常来说：
- 对手已立直时，不要轻易开杠

因为杠会：
- 给立直家增加里宝牌收益机会
- 增加自己防守难度
- 放大放铳损失

只有在你也已高质量听牌、
且杠后收益明显时才可考虑例外。

## 5. 门清大明杠几乎都要谨慎

从门清直接大明杠，
往往会暴露信息并改变攻防节奏。

若不能直接带来明显优势，
通常不推荐这么做。

## 小结

“能杠”不等于“该杠”。

实战判断顺序：
1. 自己手牌是否领先
2. 杠后受入是否受损
3. 场上是否已高压（立直/冲突局）
4. 杠收益是否足以覆盖风险

四项里有两项不利时，通常应放弃开杠。
