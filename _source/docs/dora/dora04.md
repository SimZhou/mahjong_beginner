# 宝瓷砖切割时机

我想一直困扰我的是何时切割宝藏瓷砖。

不能太早，也不能太晚。

## 角色牌是宝藏牌的情况

听到最麻烦的就是宝藏牌，这是角色牌。

什么时候是最佳修剪时间？如果你问我

回答**第一次射击**或**木偶时间**。

出于某种原因，第一枪更好。

这是因为，刚开始时，第一枪是最难击中的。

例如，如果最后一局游戏的牌是这样发的，
